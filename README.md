# resume
个人简历生成

## 推荐阅读：[如何书写一份好的互联网校招简历](https://juejin.cn/post/6928390537946857479)

## 如何食用
1. clone仓库代码
```shell
git clone https://github.com/ATQQ/resume.git
```
```shell
cd resume
```
2. 安装依赖
```shell
npm install
```
3. 本地运行
```shell
npm run dev
```
4. 构建
```shell
npm run build
```
